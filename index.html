Angular Tiles

The Window Tile Manager for AngularJS

Angular Tiles provides a system to easily create tiling layouts with AngularJS using custom tags and element attributes.

Getting Started

  1. Download the release files.
  2. Add AngularTiles as a module to your AngularJS app: 
    <script type="text/javascript">
    angular.module("myAngularApp", ["angular-tiles"]);
</script>
  3. Follow the Basic Template to get started, then the section on Tile Layout for more advanced features.

Basic Template

<!-- Start with an at-container (setting tile-fullscreen="false" forces the container to be it's parents dimensions) -->
<at-container tile-fullscreen="false">
    <!-- then nest any number of at-tile elements -->
    <at-tile tile-height="40px">
        <div class="header">
            <h3>Angular Tiles</h3>
        </div>
    </at-tile>
    <!-- at-tile elements must be direct children of an at-container or at-tile --> 
    <at-tile tile-overflow="auto">
        <div class="content">
            ...
            Lorem Ipsum Content Here
            ...
        </div>
    </at-tile>
    <at-tile tile-height="40px">
        <div class="footer">
            <ul class="nav nav-pills nav-justified">
                <li><i class="glyphicon glyphicon-chevron-left"></i></li>
                <li>MENU</li>
                <li><i class="glyphicon glyphicon-chevron-right"></i></li>
            </ul>
        </div>
    </at-tile>
</at-container>
